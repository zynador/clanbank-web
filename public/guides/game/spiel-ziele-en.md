# Game and Goals

While this is fundamentally a war game focused on building strength, a solid foundation is needed first. The goal should be unlocking T4 units and a Royal Combat Set in Gold.

## Combat Strength

Combat strength is mainly influenced by your combat set, investments, gems, crystals, and collections.

Commit to a Royal Combat Set early and build it up. The Victor Set is useless. For the building phase, a Junior Set at Gold level is recommended — it requires no contracts and already provides solid attack values.

Once attack values are solid, quickly leveling HP is important. HP values decide many battles — attack values are actually secondary.

## Troops

Power rating is just a number and says nothing about actual strength.

Until T4 is unlocked, training 400,000 units of each troop type is sufficient (Cars only need 50,000). This ensures all troops fit in hospitals and the Sanatorium.

T1 Bikers for fast RSS station raids are also recommended.

Only once T4 is unlocked does the real troop build-up begin. Even then: match troop count to hospital capacity. Below 500% attack values: only train enough T4 for raid participation — 600,000 to 800,000 of each is a good foundation.

A meat shield of T3 units can be helpful — they get killed first, protecting your T4.

## Investment

Priority: Economy → Armament → Defense.

The "a little bit of everything" approach does not work — it takes too long. Focus on the investments listed above. Then quickly level Master Armament to reduce training costs.

## Gems

Fill all slots with purple gems first. Only upgrade to gold gems once all purple slots are filled — three purple gems (3 × 9%) provide more value than one gold gem (15%). If in doubt, break down a gold gem into purples.

## Governor Battle (GOV)

We play GOV for fame and bonuses. Different player types are needed:

- Raid Leaders: attack values above 1,300% in combat analysis
- Raid Fillers: fill with T4, own stats don't matter — but must be able to withstand a raid on their villa
- Farmers/Build Teams: stay protected in clan territory, gather RSS for fighters

## Villa Protection

First rule for fighters: activate ANTI-SCOUT.

In the building phase: hide and redeploy troops. With higher troop counts, raids must be "taken". Then: equip combat set, defense item, set talents to defense, request reinforcements or port far away.

The UB trap is common in GOV: an enemy UB is sent with T1 Bikers to an RSS station so you capture them, making you attackable. Bubble during raids is then advisable.

General rule: stay in your combat set when going offline without a bubble — so clan members know how to reinforce you.
